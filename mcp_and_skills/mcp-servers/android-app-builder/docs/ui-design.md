# UI Design for Android
Source: https://developer.android.com/design/ui

## Design for Mobile
Create beautiful, usable modern app designs using Android themes, components, and adaptive layouts.

## Adaptive Design
Create immersive UX with layouts that resize and reconfigure for multiple form factors:
- Mobile phones
- Tablets and large screens
- Wear OS smartwatches
- Android TV
- Android for Cars
- XR Headsets and Glasses
- Widgets
