## How to use these links in practice

- When unsure about current best practice or API shape, prefer checking these docs over memory.
- If a suggestion depends on Gradle/AGP behavior, consult the Gradle build overview + Gradle API ref.
- If a request involves AI features, start with Gemini in Android and the Android AI hub and design for
  failure modes (offline, rate limits, missing model, privacy constraints).
